QUICK SETUP (5 Minutes)
================================================================

Step 1: Install Backend Dependencies
-------------------------------------
cd /path/to/FINAL_PROJECT
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

Step 2: Install Frontend Dependencies
-------------------------------------
cd frontend
npm install
cd ..

Step 3: Start Backend Server (Terminal 1)
-------------------------------------
cd backend
python app.py

Backend will run on: http://localhost:5000

Step 4: Start Frontend (Terminal 2)
-------------------------------------
cd frontend
npm start

Frontend will open on: http://localhost:3000

Step 5: Test the Application
-------------------------------------
1. Open browser to http://localhost:3000
2. Upload an image with a license plate
3. Click "Recognize Plate"
4. View results!

TROUBLESHOOTING
================================================================

Backend won't start:
- Check if port 5000 is available
- Verify Python dependencies: pip install -r requirements.txt
- Check Python version: python3 --version (need 3.8+)

Frontend won't start:
- Check if port 3000 is available
- Reinstall: rm -rf node_modules && npm install
- Check Node version: node --version (need 14+)

No plate detected:
- Ensure image has clear license plate
- Image should have good lighting
- Works best with Indian-style plates

Import errors:
- Activate virtual environment: source venv/bin/activate
- Reinstall requirements: pip install -r requirements.txt --upgrade
